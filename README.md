# 德州撲克計時器 (Poker Timer)

一個專為德州撲克比賽設計的多計時器網頁應用，支援自訂盲注級別、時間編輯和多桌同時計時。

## 🎯 功能特色

### 核心功能
- **多計時器支援**: 同時運行最多 4 個獨立計時器，適合多桌比賽
- **盲注自動升級**: 自動切換盲注級別，並播放提示音效
- **可視化時鐘編輯器**: 通過拖動時鐘指針直觀地調整時間進度
- **快進功能**: 一鍵跳轉到下一個盲注級別
- **自訂盲注結構**: 完全自訂每個級別的小盲/大盲/時長

### 操作按鈕
- **開始** (綠色): 啟動計時器
- **暫停** (橙色): 暫停計時
- **重置** (紅色): 重置計時器到初始狀態
- **快進** (紫色): 跳到下一個盲注級別
- **編輯** (深橘色): 開啟時鐘編輯器，調整當前時間
- **設定** (藍色): 修改盲注級別配置

## 📋 默認盲注結構

```
級別 1: 小盲 100 / 大盲 200 (60分鐘)
級別 2: 小盲 0 / 大盲 0 (10分鐘) - 休息時間
級別 3: 小盲 200 / 大盲 300 (30分鐘)
級別 4: 小盲 200 / 大盲 400 (30分鐘)
```

## 🚀 使用方法

### 基本操作
1. 點擊「開始」按鈕啟動計時器
2. 計時器會顯示當前盲注級別和剩餘時間
3. 時間到達 0 時自動進入下一級別並播放提示音
4. 可隨時「暫停」或「重置」計時器

### 自訂盲注級別
1. 點擊「設定」按鈕
2. 在文本框中按格式輸入：`小盲/大盲/時間(分鐘)`
3. 每行一個級別，例如：
   ```
   100/200/60
   0/0/10
   200/300/30
   200/400/30
   ```
4. 點擊「保存」按鈕

### 使用時鐘編輯器
1. 點擊「編輯」按鈕開啟時鐘編輯器
2. 用滑鼠拖動綠色指針調整時間
3. 即時顯示調整後的時間
4. 點擊「確認」應用修改，或「取消」放棄更改

### 快進功能
- 點擊「快進」按鈕立即跳到下一個盲注級別
- 適用於需要調整比賽進度的情況

## 🛠️ 技術架構

### 前端技術
- **HTML5**: 結構化網頁內容
- **CSS3**: 響應式設計，支援桌面和移動設備
- **Vanilla JavaScript**: 無框架依賴，輕量高效

### 核心功能實現
- **計時器類別 (PokerTimer)**: 物件導向設計，每個計時器獨立運作
- **SVG 時鐘**: 使用 SVG 繪製可互動的時鐘介面
- **事件驅動**: 滑鼠/觸控事件處理，支援拖動操作
- **Web Audio API**: 生成盲注升級提示音效

### 瀏覽器支援
- Chrome/Edge (推薦)
- Firefox
- Safari
- 支援觸控設備

- **黑色背景**: 降低眼睛疲勞，適合長時間使用
